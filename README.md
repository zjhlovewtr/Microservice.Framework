# Microservice.Framework
一、简介
aspnetcore3.1-微服务框架，以特性驱动的微服务框架
该微服务框架主要解决微服务通信和微服务集群的问题，帮助大家快速落地微服务项目
特征：
1、中台调用组件，
2、断路器Polly组件，
3、注册中心组件，
4、负载均衡组件，
5、动态中台调用组件，
6、微服务客户端组件，
7、框架异常组件
